# Rota com Família

Site institucional da **Rota com Família** — viagens, milhas, cartões e tecnologia para famílias que querem ver o mundo sem pesar no bolso.

**🎥 Canal no YouTube:** [@Rotacomfamilia](https://www.youtube.com/@Rotacomfamilia)
**📸 Instagram (Kharol):** [@kharol.antunes](https://www.instagram.com/kharol.antunes/)
**🎵 TikTok:** [@rotacomfamilia](https://www.tiktok.com/@rotacomfamilia)

## Stack

Site estático em HTML + CSS + JavaScript puro, com Tailwind CSS via CDN e animações customizadas. Pronto para hospedagem em qualquer servidor de arquivos estáticos (GitHub Pages, Netlify, Vercel, etc.).

## Estrutura

```
index.html    # marcação completa do site (uma página única)
styles.css    # componentes e identidade visual (ouro mostarda + oliva)
script.js     # animações reveal, scroll progress, contadores, modais
```

## Desenvolvimento

Não há build step. Basta abrir `index.html` em qualquer navegador moderno.

## Deploy

### GitHub Pages
1. Settings → Pages → Build from branch → `main` → `/ (root)`
2. URL pública: `https://jeffersonarrudacwb.github.io/rotacomfamilia/`

### Custom domain
Adicionar arquivo `CNAME` com `rotacomfamilia.com` e configurar DNS apontando para os IPs do GitHub Pages.
